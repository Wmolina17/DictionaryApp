## Instalación

Clona el repositorio y accede a la carpeta del proyecto:
```bash
git clone <URL_DEL_REPOSITORIO>
cd nombre-del-proyecto
```

Instala las dependencias:
```bash
npm install
```

## Ejecución
Para iniciar el servidor de desarrollo, ejecuta:
```bash
npm run dev

## Tecnologías utilizadas
